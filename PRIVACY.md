# Tabpane Privacy Policy

Last updated: June 30, 2026

Tabpane is a Chrome extension for viewing, searching, organizing, and saving browser tabs.

## Data Tabpane Handles

Tabpane can read the following information from your browser so it can show and manage your open tabs:

- Tab URLs
- Tab titles
- Favicons
- Window and tab group metadata
- User-entered window names and saved-session names

When you save a session, Tabpane stores the saved web tab URLs and titles so you can reopen them later.

## How Data Is Used

Tabpane uses this data only to provide its tab management features:

- Display open tabs and windows
- Search and group tabs
- Move, focus, close, and reopen tabs at your request
- Save and restore tab sessions
- Sync saved sessions through Chrome Sync when Chrome Sync is enabled

## Storage And Sync

Window names are stored locally on your device using Chrome extension storage.

Saved sessions are stored using Chrome's built-in `chrome.storage.sync` when available. This lets saved sessions follow the same Chrome profile across signed-in Chrome browsers when Chrome Sync is enabled. If Chrome Sync is unavailable, Tabpane keeps a local fallback copy on that device.

Tabpane does not operate its own servers and does not send your tab data to Tabpane or any third-party service.

## Sharing And Sale Of Data

Tabpane does not sell user data.

Tabpane does not transfer user data for advertising, personalized ads, retargeting, creditworthiness, or unrelated analytics.

Tabpane does not allow humans to read user data.

The only sync-related transfer is handled by Chrome Sync as part of the browser's built-in storage functionality when the user has enabled Chrome Sync.

## Remote Code And Analytics

Tabpane does not execute remote code.

Tabpane does not include analytics, tracking, ads, affiliate links, or telemetry.

## Data Retention And Deletion

Saved sessions remain until you delete them in Tabpane or remove the extension data from Chrome.

You can delete saved sessions from the Sessions panel. You can also remove all Tabpane data by removing the extension or clearing the extension's storage from Chrome.

## Contact

For privacy questions, contact the publisher using the support contact listed on the Chrome Web Store listing.
