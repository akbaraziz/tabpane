## Short Description

See, search, organize, and save open tabs with window names, Chrome tab groups, duplicate cleanup, and synced sessions.

## Detailed Description

Tabpane turns crowded browser windows into a clean, searchable board for your open tabs.

Instead of hunting through a strip of tiny favicons, open Tabpane to see every tab laid out by window with readable titles, URLs, favicons, tab counts, and Chrome tab group context.

What Tabpane helps you do:

- See all open tabs across your browser windows
- Search tabs instantly by title or URL
- Click any tab to jump straight to it
- Drag tabs between windows
- Name browser windows inside Tabpane
- View Chrome tab group names and colors
- Rename Chrome tab groups from Tabpane
- Save tab sessions and reopen them later
- Sync saved sessions through Chrome Sync when Chrome Sync is enabled
- Restore saved Chrome tab groups into their own grouped windows
- Find and close duplicate tabs
- Spot tabs that are playing audio

Tabpane is built for people who keep many tabs open while researching, reviewing, planning, building, shopping, or working across multiple projects.

Privacy:

Tabpane runs in your browser. It does not include ads, analytics, tracking, affiliate links, telemetry, or remote code. Tabpane does not operate its own servers and does not send your tab data to Tabpane or third-party services.

Saved sessions use Chrome's built-in sync storage when Chrome Sync is enabled for your profile. If Chrome Sync is unavailable, Tabpane keeps saved session data locally on that device.

## Single Purpose

Tabpane is a visual tab manager that helps users view, search, organize, save, sync, and reopen browser tabs.

## Permission Justifications

tabs:
Required to read tab titles and URLs, display open tabs, focus selected tabs, move tabs between windows, close duplicate tabs at the user's request, and reopen saved sessions.

tabGroups:
Required to display Chrome tab group names and colors, rename Chrome tab groups from Tabpane, and recreate saved tab groups when restoring sessions.

storage:
Required to save user-created window names and saved tab sessions. Saved sessions use Chrome Sync storage when Chrome Sync is enabled, with local fallback when sync is unavailable.

favicon:
Required to display page favicons through Chrome's built-in favicon API without loading remote favicon images directly from websites.

- Location: No

Data use:
